# Lab Context

## Purpose
This folder is the active development environment for writing original code, running tests, experimenting with new ideas, and finalizing implementations.

## Mindset & Rules
- **Iterative & Robust**: Focus on high-quality implementation, clean code practices, and rigorous testing.
- **Action-Oriented**: This is the place to break things, experiment, build out features, and fix bugs.
- **Self-Contained**: Ensure that the code written here is well-documented, modular, and cleanly separates experiments from final implementations.
