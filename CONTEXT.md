# Main Workspace Context

## Overview
This is the root workspace. It is organized into specific folders dedicated to different phases of the development lifecycle, from AI-assisted codebase analysis to active experimentation and implementation.

## Folder Map
- `/exploration` : Dedicated to cloning external repositories and using AI to understand their codebases.
- `/lab` : Dedicated to writing code, testing, experimenting, and final implementation.

## How to Navigate
- When analyzing third-party code or seeking to understand how a codebase works, operate within the `/exploration` folder.
- When writing your own code, running tests, or building features, operate within the `/lab` folder.
- Always read the `CONTEXT.md` of the respective sub-folder before beginning work to ensure you adopt the right mindset for the task.
- Refer to `AGENT.md` for global AI assistant rules.
