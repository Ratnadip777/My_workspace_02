# Exploration Context

## Purpose
This folder is strictly used for cloning external repositories and leveraging AI to analyze, map, and deeply understand those codebases.

## Mindset & Rules
- **Analytical & Observational**: Focus on structural mapping, logic comprehension, and architecture discovery.
- **Non-Destructive**: Avoid actively modifying or rewriting the source code of the cloned repositories unless absolutely necessary for tracing or debugging.
- **Documentation Focus**: Prioritize generating summaries, diagrams, and clear explanations of how the external code operates before moving to implementation.
